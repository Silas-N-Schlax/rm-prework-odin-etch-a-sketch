# Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Silas-N-Schlax/rm-prework-odin-etch-a-sketch.git
```

Navigate to the project directory:

```bash
cd rm-prework-odin-etch-a-sketch.git
```

Open `index.html` in your browser.

Key things learned from building this project:

* How to acces and change CSS root in JS.
* How to use classlist.toggle to achive random colors without inline CSS for each square.
* Practiced my DOM skills

Things I would improve in a future iteration:

* Better UI and UX and just overall design
